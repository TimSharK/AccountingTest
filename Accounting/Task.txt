Тестовое задание .NET 


1. Платформа: IIS, ASP.net, C# 
2. СУБД: MS SQL Server 
3. Архитектура: n-layers (Presentation, Business and Data Layer) 

Задание: создать веб-приложение для учета выкладки техники в торговой точке. 
Веб-приложение представляет собой три страницы: 
1. Страница авторизации 
2. Страница просмотра отчета по выкладке 
3. Страница администратора 
Описание страниц: 
1. Страница авторизации 

Необходимо реализовать страницу с авторизацией пользователей 
2. Страница просмотра отчета по выкладке 

Представляет собой редактируемый datagrid c полями (колонками): 
Наименование Техники 
Дата проверки 
Параметры выкладки (представляют собой отдельные колонки, настраиваемые администратором в кол-ве от 1 до бесконечности) 

3. Страница администратора 

Предложить вариант добавления дополнительных параметров выкладки для 
динамического формирования отчета на странице 2. 
Параметры могут быть вида: bool, string, double, file (например, фотографии полок с видеотехникой)
